# Benjamin Moore Store Locator Data

All Benjamin Moore retail locations in the United States, scraped from their store locator API.

## Files

- **bm_stores.csv** — 7,021 stores with full details (name, address, phone, email, hours, services, brands, products, coordinates, etc.)
- **bm_map.html** — Interactive map of all locations (open in browser)

## Data Fields

Store Number, Name, Address, City, State, Zip, Phone, Email, Website, Hours (Mon-Sun), Open 7 Days, Services, Brands, Products, Signature/Featured/Specialty Store flags, Latitude, Longitude, URL

## Coverage

~93% of the 7,578 total US stores. Some stores in very dense metro areas may be missing due to API pagination limits (max 10 results per query).

## Date

Scraped: March 3, 2026
